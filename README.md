# 🚀 Azm | عزم
تطبيق ذكي يساعدك تتجاوز صعوبة البداية، ويحول التسويف إلى خطوات تنفيذ فعلية باستخدام الذكاء الاصطناعي.

---

## ✨ الفكرة

“عزم” مو مجرد شات بوت…
هو نظام كامل يبدأ معك من:

1. فهم حالتك
2. دعمك نفسيًا
3. تقسيم المهمة
4. مساعدتك تبدأ فعليًا

🎯 الهدف: **كسر حاجز البداية**

---

## 🧠 كيف يشتغل؟

### 1. إدخال المهمة

المستخدم يكتب المهمة اللي ما يقدر يبدأ فيها.

---

### 2. تحليل الحالة

يتم استخدام نموذج لغوي (LLM) لفهم سبب التسويف مثل:

* الخوف
* الكمالية
* التردد
* الإرهاق

---

### 3. دعم مخصص

“عزم” يرد بأسلوب إنساني بسيط ويحفّز المستخدم.

---

### 4. تقسيم المهمة

تحويل المهمة إلى:
👉 Micro Tasks (خطوات صغيرة جدًا)

---

### 5. التنفيذ (Spark Mode 🔥)

* زر "شرارة العزم"
* مؤقت 7 دقائق
* يساعدك تبدأ بدون ضغط

---

### 6. التقدم

* Checkbox لكل خطوة
* الانتقال خطوة خطوة
* متابعة الإنجاز

---

## 🎮 ميزات التطبيق

* 💬 محادثة ذكية بالعربي
* 🧠 تحليل التسويف
* 📋 تقسيم المهام
* ⏱ مؤقت تركيز (7 دقائق + 3 دقائق)
* 🔥 شرارة العزم (Flame Timer)
* ✅ نظام خطوات تدريجي
* 🎯 نقاط تحفيزية (Gamification)
* 💾 حفظ التقدم (Local Storage)
* 🌙 دعم Dark Mode

## ⚙️ التشغيل

### 1. تثبيت المتطلبات:

```bash
pip install -r requirements.txt
```

---

### 2. تشغيل السيرفر:

```bash
uvicorn main:app --reload
```

---

### 3. فتح الموقع:

```
http://localhost:8000
```

## 🌐 API Endpoints

### 📩 /chat

يرسل رسالة للـ AI

```json
POST /chat
{
  "messages": [...],
  "profile": {}
}
```

---

### 📋 /steps

تحويل المهمة إلى خطوات

```json
POST /steps
{
  "task": "ذاكر رياضيات"
}
```
