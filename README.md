## 🧠 وصف المشروع
**عزم** هو مساعد ذكي مبني باستخدام الذكاء الاصطناعي يساعد الطلاب على التغلب على التسويف والبدء في مهامهم بطريقة بسيطة ومنظمة.

يعتمد المشروع على دمج:
- تحليل نفسي بسيط لسلوك المستخدم
- واجهة تفاعلية عبر Streamlit
- نموذج لغوي ذكي للتفاعل مع المستخدم
- نظام تحويل المهام إلى خطوات تنفيذ صغيرة

---

## 🎯 فكرة المشروع
غالبًا سبب التسويف ليس الكسل، بل:
- تضخيم حجم المهمة
- الخوف من الفشل
- الكمالية
- الإرهاق
- عدم وضوح نقطة البداية

يقوم "عزم" بـ:
1. فهم حالة المستخدم من خلال النص
2. الرد بأسلوب إنساني داعم
3. تحويل المهمة إلى خطوات صغيرة جدًا
4. دفع المستخدم للبدء مباشرة
